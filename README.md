## Installation

### Conda

```bash
conda env create -f environment.yml
```

### Local

```bash
pip install -e .
```

or

```bash
pip install -r requirements.txt
```


## Reproduce Our SOTA Results
```
python run.py --config configs/ours.yaml
```
